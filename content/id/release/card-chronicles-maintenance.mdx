<Section icon={Target} title="Cara Bersiap Menghadapi Downtime Terjadwal">

Untuk meminimalkan dampak **pemeliharaan Card Chronicles** pada progres Anda, Anda harus mengikuti rutinitas tertentu sebelum server mati. Karena game ini berpusat pada rolling dan manajemen sumber daya, terjebak di tengah sesi bernilai tinggi bisa sangat membuat frustrasi.

<Steps>
  <Step num="1" title="Hentikan Penggunaan Ramuan (Potion)">
    Hentikan aktivasi Luck Potion, Roll Speed Potion, atau Super Luck Potion jika hitung mundur pemeliharaan kurang dari durasi peningkatan (boost) tersebut.
  </Step>
  <Step num="2" title="Selesaikan Wave Saat Ini">
    Selesaikan dorongan wave Anda saat ini. Hadiah untuk progres wave biasanya disimpan setelah menyelesaikan suatu tahap, jadi jangan memulai pertarungan bos yang panjang tepat sebelum server ditutup.
  </Step>
  <Step num="3" title="Tinjau Barisan (Lineup) Anda">
    Pastikan kartu terkuat Anda terpasang di lineup utama Anda. Meskipun data dicadangkan, mengaktifkan tim utama Anda adalah tindakan pengamanan yang baik.
  </Step>
  <Step num="4" title="Periksa Reroll Trait">
    Jika Anda memiliki Reroll Trait yang tertunda, gunakan atau simpan. Hindari mengklik "Reroll" di detik-detik terakhir untuk mencegah "ghost roll" di mana sumber daya habis tetapi trait tidak diperbarui.
  </Step>
</Steps>

<Callout type="tip" title="Tip Pro">
Jika Anda sedang melakukan roll saat server ditutup, game biasanya menyimpan roll sukses terakhir Anda. Namun, selalu lebih aman untuk berhenti melakukan roll secara manual saat penghitung waktu mencapai tanda 1 menit.
</Callout>

</Section>
